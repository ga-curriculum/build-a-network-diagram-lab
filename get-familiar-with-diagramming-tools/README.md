**Learning objective:** By the end of this lesson, students will be able to navigate the basic features and functions of draw.io for creating network diagrams.

Before we dive into creating our network diagram, let's get acquainted with the tool we'll be using: draw.io. Draw.io is a powerful, free, and easy-to-use diagramming application that runs in your web browser. It's perfect for creating professional-looking network diagrams.

## Exercise

1. Open your web browser and go to [draw.io](draw.io).

2. Take a few minutes to explore the interface. Familiarize yourself with the main sections:

   - The toolbar at the top, which contains tools for creating and editing shapes.

   - The sidebar on the left, which contains various shape libraries.

   - The main canvas in the center, where you'll build your diagram.

3. Experiment with the following basic functions:

   - Dragging and dropping shapes from the sidebar onto the canvas.

   - Connecting shapes with lines using the connector tool.

   - Adding text to shapes and lines.

   - Moving, resizing, and deleting shapes.

4. Explore the "Network" shape library in the sidebar. This contains many of the common shapes you'll use in your network diagram, like routers, switches, servers, and cloud symbols.

5. If you're new to draw.io, consider watching a basic tutorial video to get a more guided introduction. The [draw.io YouTube Beginner Playlist](https://www.youtube.com/playlist?list=PLX6xdk86h_0xpW82Q0YkdN6xpHa6hvHjO) has some great resources.

Take your time getting comfortable with the tool. In the next exercise, we'll start thinking about the specific components of your network that you'll need to represent in your diagram.

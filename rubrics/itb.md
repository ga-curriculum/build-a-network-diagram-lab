## Lab requirements

The following requirements describe the necessary deliverables for the Build a Network Diagram Lab.

| Criteria                                                                                                                      | Points |
| ----------------------------------------------------------------------------------------------------------------------------- | ------ |
| The submitted network diagram is a PDF file.                                                                                  | 10     |
| The diagram accurately includes at least 80% of the components in the network.                                                | 40     |
| The diagram accurately represents at least 80% of the connections between components of the network.                          | 15     |
| A knowledgeable individual viewing the network diagram for the first time is able to understand the structure of the network. | 20     |
| Visual elements and design in the network diagram are professional and components are accurately represented.                 | 15     |
| **Total possible points**                                                                                                     | 100    |
